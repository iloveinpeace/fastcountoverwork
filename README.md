# 快速計算時數

這是一個可以直接部署到 GitHub Pages 的純前端網頁工具，用來快速計算員工打卡的實際工時與每日加班時數。

## 功能

- 手動輸入員工、日期、上下班時間、休息分鐘
- 支援快速輸入格式，例如 `0401,0900,1800`
- 自動計算實際工時
- 依台灣常見需求，以每日超過 8 小時的部分顯示為加班
- 同一員工同一天多筆資料可自動彙總
- 支援匯出 CSV，Excel 可直接開啟
- 使用 `localStorage` 暫存，不需要後端資料庫

## 本機使用

直接打開 `index.html` 即可。

如果你想用本機伺服器預覽，也可以在這個資料夾執行：

```bash
python3 -m http.server 8080
```

然後打開 `http://localhost:8080`

## 部署到 GitHub Pages

1. 建立 GitHub repository
2. 把這個資料夾內容上傳到 repository 根目錄
3. 到 GitHub 的 `Settings` -> `Pages`
4. Source 選擇 `Deploy from a branch`
5. Branch 選擇 `main`，資料夾選 `/root`
6. 存檔後等幾分鐘，GitHub 會提供公開網址

## 快速輸入格式

```text
0401,0900,1800
4/1,09:00,18:00
```

格式說明：

- 第一段是月日，例如 `0401`
- 第二段是上班時間，例如 `0900`
- 第三段是下班時間，例如 `1800`
- 如果上方姓名欄位有填，快速輸入加入的資料會自動帶入該姓名

## 注意

- 目前加班邏輯先以「每日工時超過 8 小時」做基礎計算
- 若你的公司有更細的休息、例假、不同班別或加班倍率規則，可以再往下擴充
